# Estuda2

Este projeto e um app de acompanhamento de estudos para concursos.

## Stack

- Next.js 15
- TypeScript
- Tailwind CSS
- shadcn/ui
- Prisma ORM
- PostgreSQL
- Zod
- React Hook Form
- Recharts

## Convencoes

- Usar Server Components por padrao.
- Usar Client Components apenas quando houver interacao.
- Validar entradas com Zod.
- Manter componentes pequenos e reutilizaveis.
- Nao criar abstracoes prematuras.
- Nao adicionar dependencias sem justificar.
- Criar codigo responsivo mobile-first.

## Criterio de conclusao

Toda feature deve:

- Compilar sem erro.
- Passar no lint.
- Ter validacao de formulario.
- Ter tratamento basico de erro.
- Ter UI responsiva.
